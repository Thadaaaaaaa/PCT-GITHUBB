เพิ่มหมวดหมู่ใหม่

🕒 ประวัติการแก้ไข

🤝 วิธีแชร์ข้อมูลให้เพื่อนร่วมทีม

1

ระบบนี้เก็บข้อมูลไว้ในเครื่องของคุณเองเท่านั้น (ไม่มี server กลาง) ดังนั้นแต่ละเครื่องจะมีข้อมูลแยกกัน

2

เมื่อแก้ไขข้อมูลเสร็จแล้ว กดปุ่ม ⬇️ Export ด้านบน เพื่อดาวน์โหลดไฟล์ .json ที่รวมข้อมูลทั้งหมดไว้

3

ส่งไฟล์ .json นั้นให้เพื่อนร่วมทีม (ผ่าน Line, Email, Google Drive ฯลฯ)

4

เพื่อนร่วมทีมเปิดเว็บนี้ในเครื่องของตัวเอง แล้วกดปุ่ม ⬆️ Import เลือกไฟล์ที่ได้รับ — ระบบจะลบข้อมูลเดิมทั้งหมดในเครื่องนั้นและแทนที่ด้วยข้อมูลจากไฟล์ทันที (ไม่ใช่การรวมข้อมูล) ดังนั้นควร Export ไฟล์ของตัวเองเก็บไว้ก่อน หากมีข้อมูลที่ยังไม่ได้ส่งให้ใคร

5

ทำซ้ำตามนี้เป็นระยะ (เช่น ทุกครั้งที่มีการอัปเดตสำคัญ) เพื่อให้ข้อมูลของทุกคนใกล้เคียงกันที่สุด — วิธีนี้ไม่ใช่การซิงค์แบบเรียลไทม์ แต่เหมาะกับการใช้งานแบบ standalone ที่ไม่ต้องพึ่ง server ภายนอก
